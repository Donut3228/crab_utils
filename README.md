# crab_utils
OpenCarbon7: utils and helpers for developers. bash_indent bash_syntax etc.

install:
mkdir -p /opt/crab/
cd  /opt/crab/
git clone https://github.com/carbonsoft/crab_utils.git
/opt/crab/utils/crab_update

usage:
crab_indent
crab_syntax
crab_example

for more info go to www.opencarbon.ru
